Pooplib

Setup

You will need:

  • The main.lua file
  • A text editor or IDE
  • Lua (5.4)

Download Lua 5.4 and then download the main.lua file.

After installation, create a new directory via the mkdir command and put the main.lua file there.

Create a new file with the .lua extenstion and paste in the following code:

M = require("main")

Now, you're done!


Pooplib functions

M.poop_text(text)

Prints text to the screen

M.set_pooparle(name, value)

Sets a variable

M.poop_text(pooparles[name])

Prints the value of a pooparle

M.add/subtract/divide/multiply(poopA, poopB)

Does maths to poopA and poopB

M.check_if_poopy(condition)

Checks if the condition is true,

Requires an if statement!



M.poop_times(times)

Prints "Poopy poop" for the number times is set to

M.poop_a_loop(starting, max, text, increment)

Prints the value of text to the screen for the value of max times. Starts at starting and end when starting is equal to max. Adds increment to starting each time.
